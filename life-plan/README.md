## ポートフォリオの改善ポイント

1. **「勝ち」はほぼ確定している水準。これからはリスクを取りすぎないことが最重要。**
   1.2億円 × 4% = 年477万円。理論上は生活費をほぼ運用益で賄える「経済的自立」に近い状態。
2. **現金が約170万円と薄い。** 生活費6か月分（300万円以上）の現金を確保し、
   さらに退職が近づいたら「生活費2〜3年分（1,500万円程度）」を現金・個人向け国債に
   段階的に移して暴落時の取り崩しリスク（収益率配列リスク）に備える。
3. **米国グロース集中ファンド（FANG+・メガ10・QQQ・NASDAQ100で約1,570万円）は攻めすぎ。**
   含み益と相談しつつ、NISA枠への移し替えの際にオルカン等へ寄せていく。
4. **月30万円の積立は「給与から」ではなく「特定口座→NISAへの移し替え」で実現する。**
   毎年、特定口座から360万円分を売却（含み益への課税は発生）し、NISA枠（年360万円）で
   買い直す。生涯枠1,800万円を最速で埋めるのが最も効率的。
5. オリエンタルランド（-56%）は優待・思い出枠として割り切るか、損出しで税還付に使うか方針を決める。
   イオン（+18%）はオーナーズカード優待があり生活費圧縮に寄与するなら保有継続で良い。

## 離婚にあたって（お金の論点）

- **財産分与**: 婚姻中に築いた資産は原則折半の対象。総資産が1.2億円規模のため影響が非常に大きい。
  離婚協議がまとまるまで大きな売買・口座間の資金移動は控え、必ず弁護士に相談する。
- **養育費**: 裁判所の算定表ベースで取り決め、公正証書にする。
- **年金分割**: 婚姻期間中の厚生年金記録の分割を忘れずに請求する（離婚から2年以内）。
- **ひとり親控除**: 所得制限（合計所得500万円以下）があるため、年収1,000万円だと対象外の見込み。
