========================================
  診療メモアプリ セットアップ手順
========================================

所要時間：約15分
必要なもの：Googleアカウント（PC使用）

----------------------------------------
STEP 1：Google スプレッドシートを作る
----------------------------------------
1. PC で https://sheets.google.com を開く
2. 左上の「＋ 新しいスプレッドシート」をクリック
3. 開いたら、上部のタイトル「無題のスプレッドシート」を
   「診療メモ」に変更する
4. このページのURLをメモしておく（後で使います）

----------------------------------------
STEP 2：Apps Script を開く
----------------------------------------
1. スプレッドシートのメニューから
   「拡張機能」→「Apps Script」をクリック
2. 新しいタブでApps Scriptが開く

----------------------------------------
STEP 3：コードを貼り付ける
----------------------------------------
【Code.gs の設定】
1. 左側に「コード.gs」というファイルがある
2. 中身をすべて削除する
3. 「Code.gs」ファイルの中身をすべてコピーして貼り付ける
4. ★重要★ ファイル上部の CLINICS の部分を
   先生の9院の名前に書き換える

   例：
   const CLINICS = [
     '〇〇歯科クリニック',
     '△△矯正歯科',
     ...
   ];

【index.html の追加】
1. 左側の「＋」ボタンをクリック→「HTML」を選択
2. ファイル名を「index」にする（.htmlは自動でつく）
3. 中身をすべて削除する
4. 「index.html」ファイルの中身をすべてコピーして貼り付ける

----------------------------------------
STEP 4：保存する
----------------------------------------
1. Ctrl + S（またはCmd + S）で保存する

----------------------------------------
STEP 5：アプリとして公開する
----------------------------------------
1. 右上の「デプロイ」ボタンをクリック
2. 「新しいデプロイ」を選択
3. 左の歯車アイコン→「ウェブアプリ」を選択
4. 以下のように設定する：
   - 説明：診療メモアプリ
   - 次のユーザーとして実行：「自分」
   - アクセスできるユーザー：「自分のみ」
5. 「デプロイ」をクリック
6. Googleアカウントの確認画面が出たら「許可」する
7. 「ウェブアプリのURL」が表示されるのでコピーする

----------------------------------------
STEP 6：iPhoneでアクセスする
----------------------------------------
1. コピーしたURLをiPhoneのSafariで開く
2. 「ホーム画面に追加」すると、
   アプリのように使えるようになる
   （画面下の「共有」ボタン→「ホーム画面に追加」）

----------------------------------------
完成！
----------------------------------------
・院名ドロップダウンから選択
・患者名を入力
・「音声入力を開始」ボタンで話すだけでメモが入力される
  （キーボードの🎤ボタンでも入力できます）
・「患者検索」タブで患者名や院名で過去のメモを検索できます

----------------------------------------
困ったときは
----------------------------------------
・URLが分からなくなった場合：
  Apps Script → デプロイ → デプロイの管理 で確認できます

・院名を変更したい場合：
  Code.gs の CLINICS の部分を書き換えて
  「デプロイ」→「既存のデプロイを管理」から更新する
