# 🎵 BGM動画ファクトリー

Suno で作った曲とジャケット画像を入れると、YouTube 投稿用の長尺BGM動画パッケージ（動画 + タイトル/説明/タグ）が出てくるローカルWebアプリ。

## 必要なもの

- Python 3.11+
- ffmpeg（`apt install ffmpeg` / `brew install ffmpeg`）
- （任意）`ANTHROPIC_API_KEY` — 設定するとメタデータを Claude AI が生成。無くてもテンプレートで動作。

## セットアップ & 起動

```sh
pip install -r requirements.txt
uvicorn app.main:app --port 8000
```

ブラウザで http://localhost:8000 を開く。

## 使い方（4ペイン）

1. **音源** — Suno の MP3 をドロップし、動画の長さ（30分〜3時間）を選択。クロスフェードで自然にループ伸長され、音量も YouTube 推奨レベルに正規化されます。
2. **ビジュアル** — ジャケット画像（ChatGPT生成など）をドロップ。ゆっくりズーム or 静止画を選択。
3. **メタデータ** — テーマとジャンルを入れて「AIで生成」。日英タイトル・説明文・タグが出るので「全部コピー」して YouTube Studio に貼り付け。
4. **キュー** — 「動画を生成する」でレンダリング開始。完了したらダウンロード。

## 注意

- 長尺（2〜3時間）のレンダリングはマシン性能により数十分かかることがあります。
- Suno の楽曲を YouTube で収益化するには Suno の有料プラン（商用利用権）が必要です。
