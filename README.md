# 確定申告シミュレーター

給与所得＋事業所得（青色申告）向けの確定申告試算ツールです。ブラウザ上で動作し、入力データはローカル（localStorage）にのみ保存されます。

## 機能

- **給与所得**: 源泉徴収票の支払金額から給与所得控除（令和7年分改正・最低保障65万円）を自動計算
- **事業所得**: 売上・経費の記帳（CSVインポート対応）、科目別集計、定額法による減価償却、青色申告特別控除（65/55/10万円）、給与との損益通算
- **所得控除**: 基礎控除（令和7年分改正の上乗せ特例対応）、社会保険料、小規模企業共済等掛金（iDeCo）、生命保険料（新制度3区分）、地震保険料、医療費、寄附金、配偶者（特別）控除、扶養控除、ひとり親・寡婦控除
- **税額計算**: 所得税（速算表）＋復興特別所得税、源泉徴収税額との差引による納付/還付額、住民税の概算（ふるさと納税の税額控除は簡略計算）

## 使い方

```bash
npm install
npm run dev    # 開発サーバー起動
npm test       # 計算エンジンのテスト
npm run build  # 本番ビルド
```

## CSVフォーマット

- 経費: `日付,区分,摘要,金額`（区分は仕入・通信費・消耗品費など。不明な区分は雑費として取込み）
- 売上: `日付,摘要,金額`

## 構成

- `src/tax/engine.ts` — 税額計算エンジン（UIから独立・テスト付き）。税制改正時はここを更新
- `src/tax/csv.ts` — CSVインポート
- `src/App.tsx` — UI

## 注意事項

本ツールは**試算用**です。住民税・寄附金税額控除は簡略計算であり、予定納税・特定親族特別控除・障害者控除・住宅ローン控除・消費税などは未対応です。実際の申告は国税庁「確定申告書等作成コーナー」等でご確認ください。
